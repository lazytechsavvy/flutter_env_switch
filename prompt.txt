🚀 Package Codename: envify (you can rename)

A lightweight, type-safe runtime environment & config switcher for Flutter apps.

⸻

🧠 1. Core Philosophy

* Zero unnecessary abstraction
* Enum-driven (no string keys for env selection)
* Runtime switching with persistence
* Minimal setup (≤ 10 lines)
* Production-safe by default

⸻

🎯 2. Scope (MVP)

MUST HAVE:

* Multi .env file loading (assets)
* Enum-based environment switching
* Runtime switching
* Persistent selection
* Simple debug panel (gesture-triggered)
* Dio integration support (optional helper)
* Soft app restart support

MUST NOT (for now):

* Remote config sync
* Encryption layer
* Over-engineered plugin system

⸻

🏗️ 3. Public API Design (FINAL)

3.1 Enum (User-defined)

enum Environment {
  future,
  staging,
  production,
}

⸻

3.2 Initialization

await Env.init(
  defaultEnv: Environment.production,
  configs: {
    Environment.future: 'assets/env/.env.future',
    Environment.staging: 'assets/env/.env.staging',
    Environment.production: 'assets/env/.env.production',
  },
);

⸻

3.3 Accessors

Env.get('BASE_URL');
Env.getInt('TIMEOUT');
Env.getBool('FEATURE_X');

Optional sugar:

Env.baseUrl

⸻

3.4 Switching

Env.switchTo(Environment.future, restart: true);

⸻

3.5 Current State

Env.current

⸻

3.6 Widget Wrapper

EnvSwitcher(
  enabled: !kReleaseMode,
  child: MyApp(),
)

⚙️ 5. Core Components

⸻

5.1 EnvConfig

class EnvConfig<E extends Enum> {
  final E env;
  final String assetPath;
  const EnvConfig({
    required this.env,
    required this.assetPath,
  });
}

⸻

5.2 EnvManager (Singleton)

Responsibilities:

* load all envs
* hold current env
* notify listeners
* persist selection

class EnvManager<E extends Enum> {
  static late EnvManager instance;
  late Map<E, Map<String, String>> _envs;
  late E _current;
  final ValueNotifier<E> currentNotifier = ValueNotifier(null);
}

⸻

5.3 EnvLoader

Loads asset files:

Future<Map<String, String>> load(String path);

Uses:

rootBundle.loadString(path);

⸻

5.4 EnvParser

Simple .env parser:

Rules:

* ignore comments (#)
* trim whitespace
* support KEY=VALUE
* support = inside values

⸻

5.5 EnvStore

Persistence layer:

Use:

SharedPreferences

Key:

'envify_selected_env'

⸻

5.6 Env (Public Facade)

class Env {
  static Future<void> init({...});
  static String get(String key);
  static int getInt(String key);
  static bool getBool(String key);
  static void switchTo(Environment env, {bool restart = false});
  static Environment get current;
}

⸻
